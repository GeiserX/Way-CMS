# CLAUDE.md — Way-CMS

## Overview
Web-based CMS for editing HTML/CSS files downloaded from the Wayback Archive. Features a file browser, syntax highlighting, live preview, multi-tenant support, and Docker deployment.

## Tech Stack
- Python (Flask)
- Nginx (static site serving)
- CodeMirror (editor)
- Docker / Docker Compose
- pytest (testing)
- Codecov (coverage)

## Development
```bash
# Run locally with Docker Compose
docker-compose up -d

# Production (uses Docker Hub image)
docker-compose -f docker-compose.prod.yml up -d

# Run tests
pytest tests/
```

## Architecture
- `cms/` — Flask application (routes, auth, models, templates, static assets)
- `nginx/` — Nginx config for serving the public site
- `tests/` — Test suite
- `scripts/` — Utility scripts
- `docs/` — Documentation and images

## Key Rules
- Docker image published to Docker Hub as `drumsergio/way-cms` with semver tags
- Never hardcode credentials; auth uses bcrypt hashing
- Production compose uses pre-built Docker Hub image, dev compose builds from source
- Multi-tenant mode (v2.0.0+) uses role-based access control and magic link auth
